# Top8000 Ramachandran Reference Data

These percentile contour grids originate from the Richardson Laboratory
`rlabduke/reference_data` repository:

- Dataset: Top8000 Ramachandran percentile contour grids
- License: Creative Commons Attribution 4.0 International
- Purpose: Ramachandran favored, allowed, and outlier classification
- Original authors: Richardson Laboratory, Duke University

The original LICENSE and README files are preserved in this directory.

EXORCIST uses Gemmi to calculate backbone torsions and performs local
lookup against these unmodified contour-grid values.
